  • Image Classification


    Technology Machine Learning
    Language Python
    Frameworks Tensorflow, CV2, TFLearn

    Aim was to predict the object of intrest in an image and give appropriate Label to it. Prepared boundry boxes to preprocess data. Compared the accuracy for algorithm with different number and type of layer, Got the highest accuray of   90.15%  with 5 layer CNN on cifar-10 dataset.

  • Catagorization of news clippings


    Technology Machine Learning
    Language Python
    Frameworks sklearn, pandas, NLTK

    Aim is to predict the appropriate catagory for the given news paper clipping, can be generalized to appropriate Tag pridiction for a blog post. compared accuracy of different algorithms like KNN, K-Means, Naive bayes and Neural-Nets with different parameters. Got maximum   99.5%  with Naive Bayes on BBC insight's database.

  • Financial Market Forecastng


    Technology Machine Learning
    Language Python
    Frameworks sklearn, pandas

    The aim was to predict the Buy/Sell signal. Used some non traditional indicators like Day of The week effect, Month of year effect and many more.

    Trying to change the problem domain to pattern recognition so that CNN can be applicable.


  • Hand Written Character Recognition


    Technology Machine Learning, Artificial Neural Network
    Language Python
    Frameworks Tensor-flow, TFLearn

    Used combination of Kaggel dataset and famous MNIST dataset to train different form of neural networks and compared accuracy, For these project I used FC Layers, RNN network, LSTM Network and CNN Networks. Got Highest accuracy of   99.2%  with CNN.
